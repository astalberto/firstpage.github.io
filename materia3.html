Estructura basica de HTML

DOCTYPE

Nos indica el tipo de documento con el que estamos trabajando, en este caso html

HTML lang

Esta etiqueta nos sirve para definir el lenguaje natural con el que vamos a trabajar, por ejemplo "en" define que trabajaremos en ingles

Head

Es el encabezado del documento HTML, donde ubicamos aquello que el usuario no vera directamente, pero, que es esencial para que funcione el sitio web

Metaetiquetas

Son elementos que se incorporan en el encabezado y que resultan invisibles para el visitante normal, pero de gran utilidad para nevegadores, diseñadores y programadores.

Atributo Descripcion Sintaxis
description Descripcion de contenido < meta name="description" content=""/>
keywords Pueden usarlas para que el motor de busqueda tenga mas informacion acerca del contenido de la pagina < meta name="keywords" content="">
charset Indica la codificacion de caracteres < meta charset="UTF-8">

Title

esta etiqueta nos sirve para dar un titulo general que es visible para el usuario, se ubica en la pestaña del navegador

link

especifica la relacion entre el documento HTML y un recurso externo

Body

Se refiere al cuerpo del documento
