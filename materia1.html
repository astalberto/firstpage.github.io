Manejadores de contenido, audio y video

Es software diseñado para facilitar la creación de páginas web estaticas

WordPress

  • Programas en creacion de bloques
  • herramienta para la creación de paginas web comerciales

Joomla

  • Es un sistema de gestion de contenidos que permite desarrollar sitios web
  • Permite CRUD de contenidos para desarrollar un sitio a traves del panel de administrador

Drupal

  • Es un sistema de gestion de contenido (CMS)
  • Modular, multipropiedad y configurable
  • Permite publicar articulos, imagenes, archivos y otros servicios como foros, encuestas, votaciones, blogs, administación de usuario y permisos

SquareSpace

  • Es un sistema de gestion de contenido, crea sitios web de diferentes tipos como blogs, o portafolios

Moodle

  • Es un software de codigo abierto
  • No se paga licencia
  • Crea perfiles de estudiantes, profesores y administrativos
