¿Qué es una direccion IP y como funciona?

Los protocolos de internet (Internet Protocol) son las anclas de internet, conectan a la red mundial y apuntalan toda su vida digital. De hecho, es imposible conectarse a internet sin una dirección IP

Analisis de las direcciónes ip

Suelen consistir en numeros del 0 al 255,separado por puntos. Dentro de una dirección IP, puede ver el ID de red y asignando a su red el ID del anfitrion, el identificador unico asignado a cada dispositivo conectado a esa red especifica
Este es un ejemplo de direccion IP: 192.168.1.3

Id de red

Es la parte de la direccion IP con la que se identifica la red que se esta utilizando para conectarse a internet es decir los 3 primeros numeros

Id de host (Anfitrion)

Una red puede ser tan pequeña como dos ordenadores conectados entre si, o tan grande como internet en si mismo, la ID del anfitrion indica el dispositivo concreto que esta usando para conectarse a una red

Ejemplo

192.168.1 3

Los numeros de color Azul representan al ID de la red y los numeros de color Amarillo representan el ID del host
