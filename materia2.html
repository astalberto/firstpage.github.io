Etiquetas

Las usamos para dar diferentes funciones a nuestro docuento html

Ejemplos

  • H1: Sirve para poner titulos en HTML

  • img: Sirve para agregar imagenes

  • a href="www.youtube.com": Sirve para crear hipervinculos como por ejemplo a youtube

  • Table: sirve para crear tablas en HTML

    a b
    1 2

  • Form: sirve para crear formularios
