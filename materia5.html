Soluciones web

Son parametros que utilizamos para medir la factibilidad de uso de una pagina web

Usabilidad

Se refiere a la facibilidad de uso que tiene una pagina u aplicacion web 3
La usabilidad mide 3 factores

  • Eficiencia: el usuario satisface la necesidad de su busqueda
  • Eficacia: el usuario logra rapidamente alcanzar su cometido
  • Satisfaccion: el usuario se sintio agradable al navegar por el sitio web

    • ¿Por que es impotante la usabilidad en una pagina web?

    la falta de usabilidad web es la mayor causa de abandono de los usuarios

Interactividad

un sitio web interactivo es aquel en el que los usuarios pueden interactuar con diferentes elementos y tener una experiencia de usuario que no se limita a hacer click y desplazarse. EStos son hechos por diseñadores y desarrolladores cuyo trabajo principal es ofrecer un sentido de personalizacion

¿Cuales son los beneficios de un sitio web interactivo?

  1. Aumentan proporcionalmente la participacion del usuario y ayudan a retener su atención
  2. Es probable que los usuarios pasen mas tiempo en el sitio web
  3. Los sitios web interactivos brindan una experiencia de usuario personalizada y ayudan a aumentar el numero de clientes satisfechos
  4. Crean un impacto positivo en los usuarios y crea un buen recuerdo

Funciones interactiva

  1. Google Maps: proporciona una buena forma de mejorar la interaccion del usuario y aumenta la confianza
  2. Crear votaciones: mostrar los resultados de las encuestas ayuda a aumentar la confianza y el compromiso
  3. Funciones de la barra de busqueda:dentro del sitio ayudan al usuario a navegar de manera mas efectiva. Funciones de filtrado de ordenacion, entre otros
  4. Calificaciones: son una forma visual rapida de obtener comentarios para mejorar productos y servicios.

Elementos esenciales

A la hora de elavorar una pagina web que sea usable debemos tener en cuenta 4 grandes elementos cuya estructura y aspecto van a determinar dicha usabilidad

  1. Identidad

    2. Es la posibilidad que refleja el sitio. Aqui es importante que la home describa adecuadamente que es lo que ofrecemos y cual es la actualidad

  2. Accesibilidad

    3. Es la posibilidad de que todos los individuos pueden hacer uso de la web, independiente de las limitaciones del individuo o dispositivo

  3. Navegacion

    4. Entendemos como tal a la facilidad de desplazarse por las distintas paginas que componen la web. El usuario debe saber en todo momento en que parte de la web se encuentra y como ir a otras secciones

  4. contenido

    5. El contenido que se ofrece sera consiso, concreto y sensillo. Debemos utilizar un lenguaje propio del publico objetivo "target" asi como ofrecer una informacion de valor para el mismo
